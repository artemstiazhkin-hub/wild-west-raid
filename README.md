# Рейд Дикого Запада

Браузерная 2D-игра в стиле тихого запада: вид сверху, стрельба по монстрам, монеты, персонажи, локации, босс и чит-коды.

## Запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
python3 -m http.server 4173
```

После этого игра будет доступна по адресу:

```text
http://localhost:4173/
```

## Файлы

- `index.html` - разметка игры
- `styles.css` - интерфейс и адаптивные стили
- `game.js` - логика игры

## Загрузка на хостинг

Для статического хостинга достаточно загрузить `index.html`, `styles.css` и `game.js` в корень сайта.
